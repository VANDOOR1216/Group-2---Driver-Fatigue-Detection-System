# 数据目录

## 📂 目录说明

此目录用于存放测试数据、示例视频和其他相关数据文件。

## 📁 目录结构

```
data/
├── videos/           # 测试视频文件
│   ├── normal/       # 正常驾驶视频
│   ├── tired/        # 疲劳驾驶视频
│   └── yawning/      # 打哈欠视频
├── images/           # 测试图片
├── results/          # 检测结果
│   ├── logs/         # 日志文件
│   └── reports/      # 测试报告
└── README.md         # 本文件
```

## 🎥 测试视频

### 视频要求

- **格式**: MP4, AVI, MOV
- **分辨率**: 建议 640x480 或更高
- **帧率**: 建议 25-30 FPS
- **时长**: 建议 10-60 秒
- **内容**: 包含清晰的人脸正面或侧面

### 视频类型

1. **正常驾驶视频** (`normal/`)
   - 驾驶员状态正常
   - 眼睛睁开，偶尔眨眼
   - 嘴巴闭合

2. **疲劳驾驶视频** (`tired/`)
   - 驾驶员疲劳状态
   - 眼睛经常闭合
   - 频繁眨眼

3. **打哈欠视频** (`yawning/`)
   - 驾驶员打哈欠
   - 嘴巴张开
   - 可能伴随闭眼

### 自采集数据

如果需要自己采集数据，请注意：

1. **隐私保护**: 获得被拍摄者同意
2. **光照条件**: 保证充足的光照
3. **拍摄角度**: 正面或侧面 ±45°
4. **视频质量**: 清晰度足够，无模糊
