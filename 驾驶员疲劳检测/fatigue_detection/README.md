# 驾驶员疲劳检测系统

## 项目概述

基于计算机视觉的驾驶员疲劳检测系统，实时监测驾驶员的面部特征，检测疲劳状态并及时预警。

## 主要功能

1. **人脸检测**：使用Dlib HOG+SVM方法实时检测驾驶员人脸
2. **面部关键点检测**：检测68个面部关键点，用于特征提取
3. **疲劳特征提取**：
   - 眼睛纵横比（EAR）：检测眨眼和闭眼
   - 嘴部纵横比（MAR）：检测打哈欠
4. **疲劳状态判定**：
   - 基于PERCLOS准则的疲劳程度评估
   - 多指标综合判断（眨眼频率、打哈欠次数等）
5. **实时预警**：根据疲劳等级发出不同级别的预警

## 技术栈

- **编程语言**：Python 3.8+
- **核心库**：
  - OpenCV：图像处理
  - NumPy：数值计算
- **开发环境**：PyCharm / VS Code
